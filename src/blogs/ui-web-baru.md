Hari ini, situs web ini mengalami perubahan UI yang signifikan, berikut perubahannya.

- Penyorotan sintaks sekarang menggunakan [`shiki`](https://github.com/shikijs/shiki).
- Semua ikon diganti dengan [`iconify`](https://iconify.design/) dengan paket [`simple-icons`](https://icon-sets.iconify.design/simple-icons/).
- Bar sisi mobile dihilangkan, sekarang semua tersedia di atas bar navigasi.
- Beranda semakin berisi.
- Dan perubahan lainnya.
